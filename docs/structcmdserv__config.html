Detailed Description

Start-up configuration for a cmdserv server.

The structure holding the configurable settings handed over to a new cmdserv server. You should prefill it using cmdserv_config_get_defaults() and then change the values you'd like to tweak.

See also:
cmdserv_config_get_defaults() cmdserv_start()

unsigned int cmdserv_config::connections_max

The maximum number of simultaneous connections the server is willing to accept.

unsigned int cmdserv_config::connections_backlog

This is the maximum number of pending incoming connections (those that are not yet accepted by the server) that may accumulate.

See your systems documentation for listen() for a more detailed explanation.

unsigned int cmdserv_config::port

The TCP port number the server should listen on.

void(* cmdserv_config::log_handler)(void *log_object, enum cmdserv_logseverity severity, const char *msg)

The callback the server will send log messages to.

The default is a built-in method that logs to STDERR.

void* cmdserv_config::log_object

A pointer to an arbitrary object that will be handed over to you again in your log_handler callback as the first argument.

Set it to NULL if you don't need that.
